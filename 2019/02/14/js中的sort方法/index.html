js中的sort方法

发表于 2019-02-14 | 分类于 js

概述

本文主要记录原生js中的sort()排序方法，跟进一步的了解sort的工作原理，方便更广泛的使用

原理介绍

  • sort() 方法用于对数组的元素进行排序，并返回数组。
  • 因为排序是按照字符串UniCode码的顺序进行排序的，所以首先应该把数组元素都转化成字符串（如有必要），以便进行比较。
  • sort()方法接收一个参数来规定排序规则，这个参数必须是函数

使用

  • 不传递参数（按字母顺序排序）
  • 传递参数（按规定规则排序）
  • 按照对象的某个属性排序
