git使用

发表于 2019-03-02 | 分类于 开发工具

概述

本文注意啊记录git的使用，通过此文章的学习，基本能满足开发中的git使用情境

git的基础命令

  • git init —初始化本地仓库
  • git add —添加文件
  • git status —查看状态
  • git commit —提交
  • git push —推送到远程仓库
  • git pull —从远程仓库拉去数据
  • git clone —从远程仓库拷贝数据

使用

安装

git 官网下载，点击安装包安装，安装完成后执行git –version可以查看到版本，则安装成功

配置

  • 配置用户：git config –global user.name <你的用户名>
  • 配置邮箱：git config –global user.email <你的邮箱>

初始化本地仓库(git init)

在项目根目录下执行：

git init 完成本地仓库初始化

添加文件(git add .)

git add . (.代表提交目录下的全部文件)

git add *.html (代表提交全部html文件)

提交文件(git comit -m “注释”)

git commit -m “这里写上你的提交注释”

至此，你的项目文件已经提交到了本地仓库中

配置不提交文件

在项目根目录下创建 .gitignore文件，在这个文件中配置好不需要提交的文件或文件夹

创建分支和切换分支

  • 创建分支： git branch 分支名称 例如：git branch login 就创建了login分支
  • 切换分支：git checkout 分支名称 例如：gitcheckout login 就将分支从原来的master分支上切换到了login分支上了

在分支中的操作在合并前不会影响到主分支的内容

合并分支（git merge）

  • 合并分支首先需要切换到主分支：git checkout master
  • 合并login分支：git merge login

连接远程仓库

git remote add origin 仓库地址

提交代码到远程仓库

  • 第一次提交

    git push -u origin master

  • 之后的提交

  • git push即可

至此，你的本地项目已经上传到了远程仓库中了

克隆（git clone）

执行git clone 远程仓库地址即可将远程仓库克隆到本地
