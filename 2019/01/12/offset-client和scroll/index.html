offset,client和scroll

发表于 2019-01-12 | 分类于 js

概述

本文主要记录offset,client和scroll三大家族的属性值代表含义和区别，在js动画中会经常使用这三个对象

offset

  • 定义

以元素的边框为界，距离具有定位父元素的偏移量，如果没有定位的父级元素，就是相对body的偏移量

offsetWidth = width + padding + border

  • 代码演示

    • 结构

      console.log(‘parent.clientTop:’+parent.clientTop) //5
      console.log(‘parent.clientLeft’+parent.clientLeft) //5
      console.log(‘parent.clientWidth’+parent.clientWidth) //320
      console.log(‘parent.clientHeight’+parent.clientHeight) //320
      `

scroll

  • 定义
    内容滚出限定区域的大小

    • scrollWidth = width + padding -(如果有滚动条)滚动条宽度
    • scrollHeight = height + padding + 隐藏内容高度
    • scrollTop和scrollLeft就是内容滚动出去的宽高
