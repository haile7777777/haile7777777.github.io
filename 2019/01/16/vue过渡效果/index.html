vue过渡效果

发表于 2019-01-16 | 分类于 vue

概述

本文主要记录vue中的组件的过渡效果实现的几种的方式。

为单个元素定义过渡效果

  • 实现步骤：

    • 将要添加过渡效果的元素使用标签包裹
    • 定义css样式

修改过渡类名前缀

  • 给transition添加name属性，就能给过渡类名修改前缀了

使用animate.css

  • 使用方法：

    • 下载animate.css，引入
    • 给标签绑定类：enter-active-class和 leave-active-class
    • 使用duration设置过渡进入和弹出的时间长度

给列表绑定过渡效果

  • 和单个元素设置过渡效果一样，只是包裹要设置过渡效果的列表用标签

  • appear属性用来初始化页面第一次加载时列表的过渡效果

  • tag属性是设置将过渡元素解析成什么元素
