概述

本系列文章主要总结前端开发中的重要知识点，包括了js基础，js高级，es6,浏览器,性能优化，vue……
这一片作为第二篇，总结js中的this指向问题和bind,call,apply方法

this指向的三种基本情况

  • 普通函数形式调用，this指向window
  • 对象的成员方法形式调用，this指向这个对象
  • 以构造函数的形式new实例，this就绑定在这个实例上，不会被任何方式改变

箭头函数的this指向

  • 箭头函数中的 this 只取决包裹箭头函数的第一个普通函数的 this
  • 对箭头函数使用 bind 这类函数是无效的

上述几种情况下的优先级

多个规则同时出现的情况，这时候不同的规则之间会根据优先级最高的来决定 this 最终指向哪里

  • new的方式优先级最高
  • 接下来是 bind 这些函数
  • 然后是 obj.foo()
  • 最后是 foo 这种调用方式

图解this指向

bind,call,apply三个方法的使用

  • bind()—第一个参数就是改变this指向的参数，后面的参数是原函数的形参列表
  • call()—第一个参数就是改变this指向的参数，后面的参数是原函数的形参列表

  • apply()—第一个参数就是改变this指向的参数,原函数的形参列表以数组的形式传入

    • call()和apply()直接调用原函数
    • bind()会复制原函数的副本返回，然后再次调用
