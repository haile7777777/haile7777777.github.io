概述

本系列文章主要总结前端开发中的重要知识点，包括了js基础，js高级，es6,浏览器,性能优化，vue……
这一片作为第一篇，总结一些js基础的知识点

js中的基本数据类型

  • 在js中原本有五种基本数据类型

    boolean ，number , string , null , undefined

  • es6中有新定义了一种数据类型

    Symbol

  • 所以，现在js中的基本数据类型包6种

小结：

  • js中的基本数据类型存储的都是值
  • number类型采用的是双精度浮点类型，所以使用中会有0.1 + 0.2 ！=0.3
  • 虽然typeOf null输出的是Object，但是，null不是对象，这是js遗留的问题

typeof和instanceof判断类型

  • typeof可以正确判断null以外的基本数据类型，判断引用类型，都是object类型

  • 判断对象的正确类型，使用 instanceof，因为内部机制是通过原型链来判断的
    基本数据类型使用instanceof不能判断，但是可以通过其他方法实现

类型转换

  • 转化为boolean类型
原始类型 转化为 结果
number Boolean 除了0，-0，NaN都为true
string Boolean 除了空字符串都为true
undefined，null Boolean false
引用类型 Boolean true
  • 转化为字符串类型
原始类型 转化为 结果
number String 5 => ‘5’
Boolean,函数,Symbol String ‘true’
数组 String [1,2] => ‘1,2’
对象 String ‘[Object Object]’
  • 转化为Number类型
原始类型 转化为 结果
String Number ‘1’ => 1 , ‘a’ => NaN
undefined Number NaN
null Number 0
数组 Number 空数组为0；数组只存在一个元素且为数字转为数字；其他NaN
其他引用类型 Number NaN
Symbol Number 抛错

小结：

  • 转Boolean: 除了 undefined， null， false， NaN， ‘’， 0， -0，其他所有值都转为 true，包括所有对象
  • “+”运算符： 运算中其中一方为字符串，那么就会把另一方也转换为字符串
    如果一方不是字符串或者数字，那么会将它转换为数字或者字符串
