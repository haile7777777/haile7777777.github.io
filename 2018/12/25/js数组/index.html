js数组

发表于 2018-12-25 | 分类于 js

概述

本文主要记录js中数组对象的操作方法，作为对数组对象的复习和查询文档。

检测数组

  • Array.isArray(value)–方法用于检测value是否是数组对象

转换方法

  • valueOf() — 返回数组本身
  • toString() — 返回以”,”连接数组各项的字符串
  • join() — 返回以指定分隔符连接的数组各项的字符串

栈方法

栈是一种”后进先出”的数据结构，对栈的操作包括入栈和弹出操作，都是对栈顶的元素进行操作。

  • js也提供了模仿栈操作的方法push()和pop()
    • push() — 将元素插入到数组末尾，返回修改后的数组长度
    • pop() — 将数组末尾的元素移出，返回移出的元素

队列方法

队列也是一种数据结构，访问规则是”先进先出”。对队列的操作是在队尾添加，队首删除

  • js提供的模仿队列操作的方法是push()和shift()
    • push()方法不再演示
    • shift()方法移除第一项并返回第一项
    • unshift()方法向数组头部添加任意个元素，并返回长度

重排序方法

  • reverse()方法 – 反转原数组，倒序输出
  • sort()方法 – 数组排序，但是sort()方法是将数组每个数据项先执行toString()方法，然后再排序，所以升序排序时”5”会排在”10”的后面，所以，想要使用sort()方法正确的排序，就要传入一个比较函数

操作方法

  • concat()方法，先创建一个当前数组副本，然后将接受参数参数添加在副本之后，返回一个新数组
  • slice()方法,从指定起始位置到指定结束位置截取数组部分元素，返回一个新数组
  • splice()方法，包含多种功能，返回一个数组，包含删除的元素，如果没有，则是空数组[]
    • 删除：传入两个参数：开始删除位置，和删除长度
    • 插入：传入三个参数：起始位置，删除长度（为0）,要插入的数据，可以是多个数据，
    • 替换：传入三个参数，开始位置，删除长度，插入元素

位置方法

  • indexOf()
  • lastIndexOf()
    这两个方法接受一个要查找的项和一个可选的起始位置参数，返回查找项的位置或者-1(表示数组中没有这一项)

迭代方法

  • every()方法，对数组的每一项传入的函数，如果该函数对数组每一项都返回true，则返回true
  • filter()方法，对数组的每一项传入的函数，返回该函数会返回true的项组成的数组
  • forEach()方法，对数组的每一项运行传入的函数，没有返回值
  • map()方法，对数组的每一项传入的函数，返回每次函数调用结果组成的数组
  • some()方法，对数组的每一项传入的函数，如果有一项返回true，则返回true

ES6中数组方法补充

Array.of()

将一组值转换为数组

copyWithin()

在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），然后返回当前数组
Array.prototype.copyWithin(target, start = 0, end = this.length)

  • target（必需）：从该位置开始替换数据。如果为负值，表示倒数。
  • start（可选）：从该位置开始读取数据，默认为 0。如果为负值，表示倒数
  • end（可选）：到该位置前停止读取数据，默认等于数组长度。如果为负值，表示倒数。

find()

  • 用于找出第一个符合条件的数组成员。
  • 它的参数是一个回调函数，所有数组成员依次执行该回调函数，直到找出第一个返回值为true的成员，然后返回该成员。
  • 如果没有符合条件的成员，则返回undefined

findIndex()

  • 返回第一个符合条件的数组成员的位置
  • 如果所有成员都不符合条件，则返回-1。

includes()

表示某个数组是否包含给定的值
